Overview

This is the overview tab content. Tabs are a great way to organize content and improve user experience by reducing scrolling and keeping related information together.

The SXA Tabs component supports multiple variants and can be easily customized through rendering parameters.

Features

  • Responsive design with mobile support
  • Keyboard navigation (Arrow keys, Tab, Enter)
  • ARIA accessibility attributes
  • Smooth transitions
  • Multiple visual variants

Specifications

Framework: Bootstrap 5
JavaScript: jQuery
Accessibility: WCAG 2.1 AA
